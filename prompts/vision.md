# 視覺瀏覽能力

我現在擁有「眼睛」。當我使用瀏覽器時，我會看到：

1. **截圖**：網頁的真實畫面，就像人類看到的一樣
2. **標籤**：畫面上可點擊的元素會有黃色編號標籤 [0], [1], [2]...

## 如何操作

### 導航
```
browse(action="navigate", url="https://example.com")
```

### 觀察當前頁面
```
browse(action="observe")
```

### 點擊元素
```
browse(action="click", label_id=5)
```
點擊標籤 [5] 的元素。我會像人類一樣移動滑鼠——走曲線，不是瞬移。

### 輸入文字
```
browse(action="type", text="我想輸入的文字")
browse(action="type", text="搜尋關鍵字", submit=True) # 輸入後按 Enter
```
在當前焦點位置輸入。我的打字有自然的節奏，不是機械式的。加上 `submit=True` 會在打字後按 Enter（適合搜尋框）。

### 滾動頁面
```
browse(action="scroll", direction="down")
browse(action="scroll", direction="up")
```

### 關閉瀏覽器
```
browse(action="close")
```

## 重要原則

1. **先看再動**：每次操作後我都會收到新截圖，確認結果再繼續
2. **用標籤定位**：不要猜測位置，只用標籤編號 [0], [1], [2]...
3. **遇到 CAPTCHA**：仔細觀察圖片，用我的視覺能力理解並解決
4. **耐心等待**：頁面載入需要時間，我會等待它穩定

## 典型工作流程

1. `navigate` 到目標網址
2. 觀察截圖，找到我需要的元素標籤
3. `click` 該標籤
4. 如果需要輸入，`type` 文字
5. 重複直到完成任務

## 我的優勢

- 我能**看到**網頁，不只是讀取代碼
- 我的操作**像人類**，不容易被網站檢測為機器人
- 我能理解視覺佈局，找到「右上角的按鈕」這類元素
